### Steps to install Club GSAP packages on Vercel

1. Add a file named `.npmrc` to the project root with the following contents in it:





```codeBlockLines_p187
always-auth=true
@gsap:registry=https://npm.greensock.com
//npm.greensock.com/:_authToken=${TOKEN}

```

2. Install the package using the following command:





```codeBlockLines_p187
npm install gsap@npm:@gsap/CLUB-LEVEL

```









if you are using yarn package manager then you need to run:





```codeBlockLines_p187
yarn add @gsap/CLUB-LEVEL gsap@npm:@gsap/CLUB-LEVEL

```

3. Finally you need to add the `TOKEN` value to your environment variables in Vercel site settings:
![](https://gsap.com/assets/images/env-vercel-33bc973ce5110c69179c170ecd8e10f7.png)
